Marco de clasificación
A Serie de acciones (workflow) — nace de orquestar pasos. Solo lógica + datos + APIs estándar. → construir en n8n/Letta/scripts.
R Exclusiva de un recurso — la otorga poseer un activo concreto (LLM, modelo visión, DB, sensor, API, credencial). Sin ese recurso, imposible. → provisionar.
A+R Mixto — un recurso ejecuta acciones (LLM + workflow, sensor + pipeline). → provisionar recurso Y construir orquestación.
N/A No replicable — biológica/corporal/subjetiva humana. → asistir (recordatorios/info) o delegar al humano.
